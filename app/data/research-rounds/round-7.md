### Detailed Research Summary

#### Round 7 – October 2024

**Research conducted:** Remote usability testing (1–2 October 2024)
**Participants:** 4 senior content designers and 1 proxy user (APHA, RPA, Countryside Stewardship)

### Key insights and recommendations by section

#### File Upload Notification Email

- **Insights:** Clear content and interaction.
- **Quotes:** "I don't know what I have to use it for, but that tells me what I need to know."
- **Recommendation:** Maintain current clarity in notification emails.

#### Creating a New Form

- **Insights:** 'Shared team email' hint text confusion regarding personal email usage.
- **Quotes:** "Personal email—my Defra email or my Hotmail?"
- **Recommendation:** Clarify the definition of 'personal email address.'

#### Adding Questions

- **Insights:** The options and previews for adding questions were clear.
- **Quotes:** "Preview is very smart!" and "Ask user to select options... most used by me."
- **Recommendation:** Continue usability testing to refine the order and clarity of options.

#### Markdown Usage

- **Insights:** Unclear if markdown usage had limitations.
- **Quotes:** "Am I limited in markdown usage, like hyperlinks?"
- **Recommendation:** Provide detailed guidance on markdown capabilities.

#### Error Messages

- **Insights:** Labels and copy clear, but interaction confusing; 'Enter' or 'Please enter' ambiguity.
- **Quotes:** "The error messages don’t work clearly."
- **Recommendation:** Standardize error message input prompts and enhance clarity.

#### User Flow and Navigation

- **Insights:** New user flow tested positively.
- **Quotes:** "It works very well!"
- **Recommendation:** Keep the revised user flow; add clear saving prompts and navigation guidance.

#### Repeater Feature

- **Insights:** Label "Repeater" clear after interaction but initially confusing.
- **Quotes:** "More direct language might help."
- **Recommendation:** Conduct further research to confirm appropriate naming and provide direct, simple examples.

#### Naming Question Sets

- **Insights:** Ambiguity around naming question sets.
- **Quotes:** "The word 'question' pops up too often and confuses."
- **Recommendation:** Revise language to plain English, direct instructions, and simplify the interaction.

#### Overview of Questions

- **Insights:** Clear overall but affirmative copy suggested.
- **Quotes:** "Text should say 'add some questions' rather than accusing users of not adding questions."
- **Recommendation:** Adjust copy to a more affirmative, positive language.

#### Adding Options for Radios

- **Insights:** Interaction and labels clear.
- **Recommendation:** Maintain current clarity; provide upfront guidance on choosing the right question type.

#### Reordering Pages

- **Insights:** Functionality clear; minor navigational friction.
- **Quotes:** "Prefer automatic return to main page after saving changes."
- **Recommendation:** Streamline navigation post-action for smoother user experience.

### Next Steps

- Revise instructional copy for shared emails and error messages
- Clarify markdown limitations and capabilities
- Refine the naming and description of complex interactions like the 'Repeater'
- Adjust affirmative copy for adding questions
- Simplify user navigation post-interaction for form editor
