Current features

The Defra Form Designer is in private beta. We're working on some exciting features right now to.

See available features below.

Form Management

Library of existing forms

Search for forms by title, keyword, author and status.

Form information

Add data about your form including submission destinations, contact details, privacy policy and organisation details.

Form Design

Page types

Question pages, guidance pages, check your answers pages, confirmation pages, and exit pages.

Error messages

Automatically generated error messages with real-time preview capabilities.

Answer limits

Set minimum and maximum character lengths or date ranges for user responses.

Form guidance

Add guidance and hint text to questions or create specific guidance pages.

Conditions

Show or skip pages based on answers. Combine 2 conditions for complex routing.

Form declaration

Include declaration text at the end of the form for users to agree to.

Information Types

Text & Numbers

  • Long or short written answers
  • Numbers
  • Dates
  • UK addresses
  • Phone numbers
  • Email addresses

File Upload

Allow users to securely upload supporting documents and images.

Selection Options

Yes or No

Simple binary choices for clear decision making.

Checkboxes

Multiple choice selections for comprehensive data collection.

Radio buttons

Single choice options from a predefined list.

Autocomplete

Smart search functionality for large datasets.

Select dropdowns

Organized lists for structured data input.

Advanced Features

Repeating questions

Get repeated sets of data from users. Design once, use multiple times.

Preview your form

Real-time preview as you build. Test layout and conditions end-to-end.
