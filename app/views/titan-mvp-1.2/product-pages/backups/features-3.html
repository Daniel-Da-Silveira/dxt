Form Management

Library of existing forms

Search for forms by title, keyword, author and status.

Form information

Add data about your form including:

  • where submission data is sent
  • contact details for support
  • what happens after someone submits the form
  • privacy policy
  • organisation details

Form design

Page types

The Defra Form Designer supports:

  • question pages
  • guidance pages
  • check your answers pages
  • confirmation pages
  • exit pages

Error messages

Error messages are automatically generated using a short description and can be previewed in real time too.

Answer limits

Limit the answers users can provide. For example, minimum and maximum character lengths or date ranges.

Form guidance

Add guidance and hint text to question or create a specific guidance page.

Conditions

Use conditions to show or skip pages based on answers. Combine 2 conditions together for complex routing.

Form declaration

Include declaration text at the end of the form for users to agree to.

Information types

Text & Numbers

You can ask users for:

  • a long or short written answer
  • a number
  • a date
  • a UK address
  • a phone number
  • an email address
  • supporting information like a file or image

File upload

Allow users to securely upload supporting documents.

Selecting from a list

Yes or no

Ask users to select answers from a list of options. Formats include:

  • Yes or no
  • Checkboxes
  • Radios
  • Autocomplete
  • Select

Advanced Features

Repeating questions

Get repeated sets of data from users with repeating questions. Design the question once and set the number of times you want users to answer it.

Previewing a form

Preview your form in real time as you build it. Use the preview link to test the layout and conditions end to end. Share a preview link with others without affecting your form.

Submitted forms

Data from a form submission is automatically sent to the shared mailbox you signed up with.

Developer Plugin

A lightweight form building tool designed for developers who want to integrate form capabilities directly into their existing applications and systems.

Key Features

Seamless Integration

Integrate forms directly into your existing applications with minimal setup and configuration.

API-First Design

Built with developers in mind, featuring comprehensive APIs for form creation, management, and data retrieval.

Custom Styling

Full control over form appearance and styling to match your application's design system.

Perfect for

  • Web applications requiring embedded forms
  • Internal tools and dashboards
  • Customer portals and self-service systems
  • Multi-tenant SaaS applications
  • Applications with custom authentication
  • Systems requiring real-time form updates
  • Applications needing custom validation rules
  • High-traffic websites and applications
