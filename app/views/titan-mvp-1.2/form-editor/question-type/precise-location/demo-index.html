Precise Location Format Demonstrations

These demo pages show how each location format appears to users, including the form fields, hint text, and error messages.

Available Location Formats

Easting and Northing
Two separate input fields for easting (0-70,000) and northing (0-1,300,000) coordinates
View demo for easting and northing format
OS Grid Reference
Single input field for Ordnance Survey grid reference (8-12 characters, e.g., TQ123456)
View demo for OS grid reference format
National Grid Field Number
Single input field for National Grid field number (10 characters, e.g., NG12345678)
View demo for National Grid field number format
Latitude and Longitude
Two separate input fields for latitude (49-60°) and longitude (-9 to 2°) coordinates
View demo for latitude and longitude format

Features Demonstrated

  • Form field layouts and styling
  • Hint text with validation ranges
  • Error messages for each validation scenario
  • Input field types and constraints
  • Accessibility features and labels
