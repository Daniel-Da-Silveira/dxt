Modular Check Answers Settings Examples

This page demonstrates how the modular components can be used to create different sub-navigation combinations.

Example 1: Page settings only

{{ checkAnswersSubNav( currentTab='page-settings', showDeclarationTab=false, showSectionsTab=false, baseUrl="/titan-mvp-1.2/form-editor/check-answers/settings-modular" ) }}

This shows only the "Page settings" tab.

Example 2: Page settings + Declaration text

{{ checkAnswersSubNav( currentTab='page-settings', showDeclarationTab=true, showSectionsTab=false, baseUrl="/titan-mvp-1.2/form-editor/check-answers/settings-modular" ) }}

This shows "Page settings" and "Declaration text" tabs.

Example 3: Page settings + Organize sections

{{ checkAnswersSubNav( currentTab='page-settings', showDeclarationTab=false, showSectionsTab=true, baseUrl="/titan-mvp-1.2/form-editor/check-answers/settings-modular" ) }}

This shows "Page settings" and "Organize sections" tabs.

Example 4: All three tabs

{{ checkAnswersSubNav( currentTab='page-settings', showDeclarationTab=true, showSectionsTab=true, baseUrl="/titan-mvp-1.2/form-editor/check-answers/settings-modular" ) }}

This shows all three tabs: "Page settings", "Declaration text", and "Organize sections".

Example 5: Declaration text tab (active)

{{ checkAnswersSubNav( currentTab='declaration', showDeclarationTab=true, showSectionsTab=true, baseUrl="/titan-mvp-1.2/form-editor/check-answers/settings-modular" ) }}

This shows the "Declaration text" tab as active.

Example 6: Organize sections tab (active)

{{ checkAnswersSubNav( currentTab='sections', showDeclarationTab=true, showSectionsTab=true, baseUrl="/titan-mvp-1.2/form-editor/check-answers/settings-modular" ) }}

This shows the "Organize sections" tab as active.

Usage Instructions

To use these modular components:

  1. Import the macro: {% raw %}{% from "components/check-answers-sub-nav/macro.njk" import checkAnswersSubNav %}{% endraw %}
  2. Use the macro with parameters:
    • currentTab: The currently active tab ('page-settings', 'declaration', or 'sections')
    • showDeclarationTab: Boolean to show/hide the declaration text tab
    • showSectionsTab: Boolean to show/hide the organize sections tab
    • baseUrl: The base URL for the navigation links
  3. Include the appropriate tab content template based on the current tab

Available Tab Content Templates

  • components/check-answers-tabs/page-settings.njk - Page settings summary list
  • components/check-answers-tabs/declaration.njk - Declaration text form
  • components/check-answers-tabs/sections.njk - Organize sections form
