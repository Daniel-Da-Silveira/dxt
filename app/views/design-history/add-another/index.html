For example, adding multiple pets with their details or multiple locations with their grid references.
Key Features and visual illustrations

The Add another feature includes:

  • Multiple responses: Allows users to add multiple items of the same type
  • Response limits: Configurable minimum and maximum number of responses (up to 25)
  • Question sets: Group related questions that can be repeated together
  • Preview functionality: Live preview of how the repeated questions will appear to users

Recent improvement: The page heading, guidance, and repeater settings are now configured on the page overview screen, rather than being asked for upfront. This gives users better context and makes the process more intuitive.
