Adding Questions

Old iteration:

  • difficult to understand different question types
  • no immediate feedback on question setup
  • complex configuration options without guidance
  • separate steps for question text and options

Latest iteration:

  • clear categorization of question types with examples
  • real-time preview of questions as they're being built
  • guided input with contextual help
  • integrated workflow for question text and options
